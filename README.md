# MiniJava Compiler

A compiler project implemented in Java using ANTLR for parsing a subset of the Java language (MiniJava).

## Features

* Lexical Analysis (Lexer)
* Syntax Analysis (Parser)
* Abstract Syntax Tree (AST) Generation
* Class Declarations
* Variable Declarations
* Method Declarations
* If / Else Statements
* While and For Loops
* Arrays Support
* Arithmetic Expressions
* Boolean Conditions
* Return Statements

## Technologies

* Java
* ANTLR
* Compiler Design
* Abstract Syntax Tree (AST)

This project demonstrates the main phases of compiler construction, including grammar design, lexical analysis, parsing, and AST generation.
